# WorthOne

פלטפורמה בעברית לריכוז כל הכספים שלך במקום אחד — עובר ושב, קופות גמל, פנסיה, השקעות ועוד — כדי לדעת בדיוק כמה אתה "שווה" ומהו ההון העצמי שלך.

הזנה פשוטה של הנכסים מציגה מיד את ההון העצמי הכולל בגרף עוגה אינטראקטיבי, שממחיש ויזואלית ממה הוא מורכב.

## עקרונות

- **פרטיות מלאה** — כל החישוב מתבצע בדפדפן שלך. שום נתון פיננסי לא נשלח או נשמר בשרת.
- **ללא הרשמה, ללא עלות.**
- **בעברית**, בעיצוב נקי (כחול־סגול על לבן).

## טכנולוגיה

HTML, CSS ו-JavaScript טהורים — ללא פריימוורק וללא שלב build. גרף העוגה מצויר ב-SVG ידני (ללא ספריות חיצוניות).

## מבנה

| קובץ | תיאור |
|------|-------|
| `index.html` | דף הבית |

| `about.html` | אודות |
| `faq.html` | שאלות נפוצות |
| `contact.html` | צור קשר |

| `js/main.js` | ניווט נייד + אקורדיון FAQ |
| `js/tool.js` | לוגיקת המחשבון וגרף העוגה |
